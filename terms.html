1. Acceptance

2. What We Collect

CounterDiscord collects the following data from servers where it is active:

  • Message metadata — message ID, channel ID, user ID, timestamp, and deletion status. Message content is never stored.
  • Emoji usage — which emojis are sent or reacted with, by which user, and when.
  • Sticker usage — which stickers are sent, by which user, and when.
  • Voice channel sessions — user ID, channel ID, join time, leave time, and duration.
  • Word hits — occurrences of server-configured tracked keywords, linked to user and channel.
  • Interaction data — reply and mention events between users (used for the social graph).
  • Server configuration — ignored channels, tracked words, and custom dashboard background images.
All data is scoped to a single Discord server (guild). Data from one server is never shared with or visible to another server.

3. How We Use Your Data

Collected data is used exclusively to generate statistics, charts, and insights displayed through bot commands within your server. It is never:

  • Sold or shared with third parties
  • Used for advertising or profiling
  • Accessed by anyone outside your server except the bot operator for maintenance

4. Data Retention

Data is retained for as long as the bot is active in your server. When the bot is removed from a server, you may request deletion of all data associated with that server by contacting the bot operator. No automatic deletion is performed upon removal.

5. User Rights

Server administrators may:

  • Exclude specific channels from tracking using /ignore-channel
  • Remove the bot at any time to stop all future data collection
  • Request deletion of all server data by contacting the bot operator

Individual users do not have a direct mechanism to delete their own data, as records are aggregated at the server level. Requests can be directed to the bot operator.
